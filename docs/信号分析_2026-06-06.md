## 什么是金叉信号？

```
MA10（10日均线）= 过去10个交易日收盘价的平均值
MA30（30日均线）= 过去30个交易日收盘价的平均值

金叉 = MA10 从下方上穿 MA30
      ↓
      短期买入力量 > 长期买入力量
      ↓
      可能意味着趋势从"下跌/横盘"转向"上涨"
```

### 为什么均线交叉有意义？

均线是市场**平均成本**的近似。

- MA10 < MA30 → 最近10天买入的人平均成本**低于**最近30天 → 价格在跌
- MA10 > MA30 → 最近10天买入的人平均成本**高于**最近30天 → 价格在涨

交叉那一刻，意味着**市场的平均成本结构正在改变**。

## 为什么300只股票只有2个金叉？

```
均线交叉是稀疏事件——不是每天都有。

在 300 只股票中：
  - 约 60% 处于 MA10 < MA30（下跌趋势）→ 不可能金叉
  - 约 30% 处于 MA10 > MA30（上涨趋势）→ 已经叉过了
  - 约 10% 处于 MA10 ≈ MA30（临界状态）→ 可能金叉
    └── 其中只有少部分恰好在今天发生上穿

今日：300只中 2 只金叉 → 0.67%，属于正常偏低。

侧面信号：如果大盘强势上涨，金叉信号应该大批出现（10-30个）。
         只有2个金叉，说明当前市场整体偏弱或震荡。
```
