### 阶段一：回测验证（第一优先级）

**目标**：用历史数据验证双均线策略的真实表现

#### 方案 A：聚宽在线回测（推荐起步）

```
为什么先选聚宽：
  ✅ 零代码搭建——浏览器里点几下就跑
  ✅ 数据齐全——自带沪深300十年数据
  ✅ 报告完整——收益率/夏普/回撤/胜率一键出
  ✅ 社区模板——直接搜"双均线"有现成代码

操作步骤：
  1. joinquant.com 注册
  2. 复制双均线策略模板
  3. 设置参数：MA10/MA30，沪深300成分股
  4. 回测区间：2020-01-01 到 2026-06-01
  5. 跑回测，看报告

关键指标观察：
  年化收益率 → 有多少？如果<5%不如理财
  最大回撤   → 最惨的时候亏多少？
  夏普比率   → 每单位风险赚多少？>1才算及格
  胜率       → 多少次交易盈利？30-40%正常
  盈亏比     → 平均赚/平均亏 >1.5才行
```

#### 方案 B：本地 Backtrader（聚宽验证通过后）

```
为什么后选本地：
  - 需要写代码搭框架（1-2天）
  - 好处：和实盘信号完全一致的逻辑
  - 好处：可以自定义任何细节
```

#### 回测必须检查的陷阱

```
□ 手续费是否设置（0.03%买卖+0.05%印花税）
□ 滑点是否设置（0.1%）
□ 是否用了未来数据（如"今天用今天才知道的财务数据"）
□ 涨跌停是否处理（涨停买不到、跌停卖不掉）
□ 是否排除了ST和停牌
□ 回测区间是否覆盖至少一轮牛熊
```

### 阶段二：策略扩展（回测通过后）

#### 策略路线图

```
现有：双均线趋势（MA交叉）
         │
    ┌────┴────┐
    ▼         ▼
动量突破    均值回归
(趋势市)    (震荡市)
    │         │
    └────┬────┘
         ▼
    多策略并行
```

#### 策略一：动量突破（强势趋势捕捉）

```
逻辑：
  今日收盘 > 过去 N 日最高收盘价 × (1 + buffer) → BUY
  今日收盘 < 过去 M 日最低收盘价 → SELL

参数：
  lookback = 20天
  buffer = 0.02（突破2%确认）

适合环境：🟢 强势市场（当前大盘择时会自动搭配）
现有框架集成：只需继承 BaseStrategy，注册即可
```

#### 策略二：均值回归（震荡市捕捉）

```
逻辑：
  收盘价 < 布林带下轨(20,2) → BUY（超卖）
  收盘价 > 布林带上轨(20,2) → SELL（超买）
  收盘价回到中轨 → 平仓

注意：
  - 只在🟡震荡市启用（大盘择时提供信号）
  - 需要趋势过滤：MA60向下时不做多
```

#### 多策略信号汇总（需要开发）

```
策略A: BUY 四川路桥, 中远海控
策略B: BUY 四川路桥, 京东方A
策略C: SELL 四川路桥
      ↓
汇总规则：
  - 同股票同方向多策略确认 → 增强置信度
  - 同股票买卖冲突 → 标记冲突，人工裁决
  - 按信号强度排序
```

### 阶段三：实盘验证（策略回测通过后）

```
前提条件：
  □ 策略在回测中表现稳定（夏普>1，最大回撤<30%）
  □ 大盘择时显示🟢强势
  □ 小资金（1手=几百到几千元）

操作流程：
  1. 收到钉钉买入信号
  2. 确认大盘是🟢强势
  3. 打开中信建投APP → 买1手
  4. 在盈亏记录里记：日期、股票、价格、止损价
  5. 每天观察钉钉止损提醒
  6. 触发止损或达到止盈 → 卖

心态：
  - 1万本金年化20% = 赚2000元（不够一顿好饭）
  - 这个阶段的目标是验证策略，不是赚钱
  - 赚了比亏了更重要的是：策略信号是否准确
```
