## [v0.2.0] - 2026-06-09

### 🚀 新增
- **防线一：大盘择时**：沪深300 + MA20/MA60 四档状态自动判断，弱势禁仓
- **每日市场日报**：宏观/板块/个股/行业四层分析，独立脚本，21:05 推送
- **信号持久化**：去重写入 signal_history，支撑复盘统计
- **交易日判断**：周末自动跳过；工作日无数据标注提示
- **性能优化**：300次 SQLite 连接→1次批量加载

- **三策略并行**：双均线 + 动量突破(v1) + 均值回归(v1)
- **信号交叉确认**：⭐⭐双确认/⭐单策略/⚠️冲突检测
- **市场-策略匹配**：强势增强趋势策略，震荡增强均值回归
- **行业板块分析**：同花顺 90 行业 + 极值过滤 + 板块内个股
- **回测系统**：三策略六方案对比（backtest/simple_backtest.py）

### 🔧 修复
- 数据下载加二轮重试+非交易日容错
- 日报数据一致性（所有组件统一 data_date）
- 指数统一日期——取交集消除错配
- 无信号时也推送
- 涨跌幅 NaN 自动补计算（fix_pct_change）
- 清理无用代码 + Mac crontab 残留修复
- 推送来源标识 📍 防多机器重复
- cron 时间多次调整：15:05 → 17:30 → 21:00
