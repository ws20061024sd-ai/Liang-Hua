## [v0.2.0] - 2026-06-09

### 🚀 新增
- **防线一：大盘择时**：沪深300 + MA20/MA60 四档状态自动判断，弱势禁仓
- **每日市场日报**：宏观/板块/个股三层分析，独立脚本，17:05 推送
- **信号持久化**：去重写入 signal_history，支撑复盘统计
- **交易日判断**：周末自动跳过；工作日无数据标注提示
- **性能优化**：300次 SQLite 连接→1次批量加载

### 🔧 修复
- 数据下载加重试+非交易日容错
- 日报数据一致性（广度统一用 DB 最新日期）
- 无信号时也推送（之前静默）
- 涨跌幅 NaN 自动补计算
- 清理无用 import 和变量
