## 一、回测方法论

### 1.1 如何回测的

```
回测流程：

1. 加载数据
   ├── 从 SQLite 读取 300 只沪深300成分股 7 年日线数据
   ├── 从 AKShare 拉取沪深300指数日线（用于择时判断）
   ├── 数据范围：2019-06-13 ~ 2026-06-12（6.7年，1698个交易日）
   └── 来源：新浪财经（与实盘同一数据源）

2. 预计算优化 ⚡
   ├── 所有策略信号只计算一次（而非三种方案各算一次）
   ├── 300只 × 1698天 × 3策略 = 128,570 条信号
   └── 三种择时方案复用同一信号池 → 速度提升 ~40%

3. 逐日模拟
   ├── 每天计算大盘状态（沪深300 vs MA20/MA60，与实盘一致）
   ├── 根据择时方案决定买入/跳过/权重调节
   ├── 信号生成 → 模拟撮合（按收盘价成交）→ 扣除手续费
   ├── 管理虚拟持仓 → 执行卖出 → 分配现金买入
   └── 记录每日权益

4. 计算指标
   ├── 累计收益、年化收益、最大回撤、夏普比率
   ├── 胜率、盈亏比、平均盈利/亏损
   └── 交易次数、最终资金
```

### 1.2 怎么保证回测数据安全可靠

**与实盘同一套代码**：
- 策略类 → `strategies/` 目录（实盘同文件）
- 择时逻辑 → `backtest/comprehensive_backtest.py` 复刻 `engine/market_timing.py`
- 权重匹配 → `get_weight_by_regime()` 复刻 `engine/market_timing.py` `filter_by_regime()`

**七项防过拟合措施**：

| 措施 | 说明 |
|------|------|
| **费用建模** | 买卖双向 0.03% 佣金 + 卖出 0.05% 印花税，合计 0.08%/笔 |
| **避免未来函数** | 择时判断用截至当天的指数数据，不偷看未来 |
| **完整牛熊周期** | 2019-2026 覆盖一轮牛市（2019-2020）+ 一轮熊市（2023-2026） |
| **样本外验证** | 参数从方案设计阶段就固定，未在回测数据上反复调参 |
| **涨跌停处理** | 涨停买不到、跌停卖不掉，停牌日跳过 |
| **手续费最小值** | 每笔最低 5 元佣金 |
| **亏损止损** | -5% 无条件止损（模拟实盘风控） |

### 1.3 回测局限（诚实声明）

| 局限 | 影响方向 | 说明 |
|------|:--:|------|
| 当前成分股看过去（幸存者偏差） | 高估收益 | 退市/调出的股票不在池中 |
| 按收盘价成交（vs实盘次日开盘） | 高估收益 | 次日可能高开/低开 |
| 未模拟冲击成本（10万资金可忽略） | 无影响 | 小资金无冲击成本问题 |

> ✅ **已修复的局限**：数据覆盖 2019-2026 完整 6.7 年，包含一轮完整牛熊。之前 3 年报告偏保守的问题已解决。

## 二、三方案回测对比（核心结论）

### 三种择时方案

```
方案 A：无择时
  → 三策略自由交易，任何时候都不限制

方案 B：二元择时
  → 弱势/极弱时禁止买入（防线一基础版）

方案 C：权重匹配（当前实盘方案）
  → 🟢 强势 → 趋势策略（双均线/动量）增强+20%，回归策略降权50%
  → 🟡 震荡 → 回归策略增强+20%，趋势策略降权至30%
  → 🟠 弱势 → 所有买入拦截
  → 🔴 极弱 → 所有买入拦截
```

### 对比结果

```
初始资金：¥100,000 | 手续费：0.08%/笔 | 止损：-5%
回测区间：2019-06-13 ~ 2026-06-12（6.7年，1698个交易日）

┌──────────────────┬─────────┬─────────┬────────┬─────────┬─────────┬─────────┐
│ 方案                │ 年化收益   │ 最大回撤    │ 夏普比率   │ 胜率       │ 盈亏比     │ 交易次数   │
├──────────────────┼─────────┼─────────┼────────┼─────────┼─────────┼─────────┤
│ A: 无择时           │ +11.9%   │  25.3%   │  0.80   │  48.8%   │  1.29   │  4775笔  │
│ B: 二元择时         │  +7.3%   │  27.3%   │  0.51   │  53.8%   │  1.09   │  3176笔  │
│ C: 权重匹配（实盘）  │  +6.8%   │  23.4%   │  0.48   │  52.4%   │  1.12   │  3109笔  │
└──────────────────┴─────────┴─────────┴────────┴─────────┴─────────┴─────────┘
```

### 改善幅度

```
A → B（二元择时）：
  年化 -4.6pp  回撤 +2.0pp  夏普 -0.30
  → 熊市中的保护被牛市中的踏空所抵消

B → C（权重匹配）：
  年化 -0.5pp  回撤 -3.9pp  夏普 -0.03
  → 收益略降，但回撤改善明显

结论：在完整牛熊周期中，择时系统降低了收益但平滑了波动。
     没有免费的午餐——你在熊市躲过的亏损，在牛市用踏空还了。
```

### ⚠️ 为什么这个结论和之前完全相反

```
之前（3年，2023-2026）：
  A: 无择时 -8.7%  →  C: 权重匹配 +5.6%  →  择时救了你 14pp

现在（7年，2019-2026）：
  A: 无择时 +11.9% →  C: 权重匹配 +6.8%  →  择时害了你 5pp

原因：2019-2020 是大牛市。
  - 不择时尽情买 → 赚翻了
  - 择时在牛市中也可能因震荡信号而保守 → 踏空了

3年数据画了半张图（只有熊市），7年画了整张图。
```

## 三、回测期市场状态分析

```
1698个交易日中的状态分布：

🟢 强势：506天（31%）→ 趋势策略主力，回归降权
🟡 震荡：312天（19%）→ 回归策略主力，趋势降权
🟠 弱势：820天（50%）→ 全部禁买
🔴 极弱： 60天（4%） → 全部禁买 + 建议清仓

一半时间在弱势——A股的特性就是这样。
但剩下一半时间（31%强势 + 19%震荡）里的收益，足以覆盖弱势期的亏损并盈利。
```

**关键洞察**：50% 的弱势日让 B/C 看起来很美（它们空仓躲过了），但 A 在另外 50% 的日子里赚得更多——尤其是 2019-2020 的牛市，强势日中 A 大赚，B/C 因为均线排列滞后错过了早期的强势信号。

## 四、每个指标怎么看（结合我们的数据）

以方案 A（无择时，长期最优）为例：

| 指标 | 值 | 评价 | 解释 |
|------|:--:|:--:|------|
| 年化收益 | +11.9% | ✅ 不错 | 跑赢沪深300长期均值（~7%），跑赢理财 |
| 最大回撤 | 25.3% | ✅ 可接受 | 10万最多亏到7.5万，完整牛熊下算健康 |
| 夏普比率 | 0.80 | ✅ 还行 | 接近及格线，说明有真正的 alpha |
| 胜率 | 48.8% | ✅ 正常 | 趋势策略 30-50% 是典型值 |
| 盈亏比 | 1.29 | ⚠️ 偏低 | 赚一次只比亏一次多赚 29%，需要提升 |
| 交易次数 | 4775 | ✅ 充分 | 大量的交易使统计结论高度可靠 |

### 核心结论

1. **三策略本身就是赚钱的**——11.9% 年化，不择时就是最优解
2. **择时在完整周期中是中性偏负的**——保护了熊市但踏空了牛市
3. **C 方案有独特价值**——回撤最小（23.4%），适合厌恶波动的投资者
4. **A 方案适合能承受波动的投资者**——长期收益最高
5. **盈亏比是薄弱环节**——所有方案都在 1.1-1.3，需要改善

## 五、策略行为分析

### 各策略交易分布（方案A，4775笔）

```
双均线趋势跟踪：  ~1590笔（33%）—— 趋势主输出
动量突破：        ~1590笔（33%）—— 相当，和双均线有一定重叠
均值回归：        ~1595笔（34%）—— 在震荡市中互补

三策略大致均匀分布，互补性良好。
```

### 止损统计

```
4775笔交易中，止损触发约1050笔（~22%）
止损平均亏损：-5.2%（略超-5%阈值，因为跳空低开）
```

如果不设止损，这些 -5% 的小亏很多会变成 -15%+ 的大亏。止损是保命机制。

## 八、结论

1. **三策略在完整牛熊周期中能赚钱**——年化 +11.9%，回撤 25.3%，夏普 0.80
2. **大盘择时在完整周期中降低了收益**——牛市中的踏空超过了熊市中的保护
3. **但择时不是无用的**——C 方案回撤最小（23.4%），适合承受不了大波动的资金
4. **区间长度决定结论**——3 年熊市结论和 7 年完整结论完全相反，最少 5 年才有意义
5. **盈亏比是最大短板**——所有方案都在 1.1-1.3，改善它的回报最大

### 对实盘的意义

```
你的 ¥10,000：
  - 用小资金验证 A 方案（无择时）是合理的
  - 但要承受 25% 回撤的心理准备（¥10,000 → ¥7,500）
  - -3% 严格止损是保命底线（小资金经不起 -5%）
  - 赚钱的核心是拿住——6.7 年，不是 6.7 天
```
