# 量化策略全景研究

> 整理时间：2026-06-13
> 来源：GitHub 开源项目 + 聚宽社区 + 券商研报 + 学术文献
> 目的：了解有什么策略可用，哪些适合我们，后续往哪个方向走

---

## 一、量化策略全景分类

```
量化策略
│
├── 方向型（赌方向，有市场风险）
│   ├── 趋势跟踪    → 追涨（动量）
│   ├── 均值回归    → 抄底（反转）
│   ├── 因子选股    → 挑好股票（价值/动量/质量/低波）
│   └── 行业轮动    → 挑好行业
│
└── 相对价值型（赌价差，对冲市场风险）
    ├── 配对交易    → 两只股票价差回归
    ├── 统计套利    → 大量股票组合对冲
    └── 纯套利      → 同一资产不同市场价格差
```

**核心思想**：价格 = 趋势 + 均值回归波动。方向型策略赚趋势的钱，相对价值型赚波动的钱。

## 二、八大策略类型详解

### 2.1 趋势跟踪

```
逻辑：涨了还会涨，跌了还会跌（惯性）
持有期：天到月
适合市场：🟢 强势趋势市
A股有效性：★★★☆ 中长期有效

代表性策略：
  - 双均线交叉（MA10/MA30）            ← 我们已有
  - 通道突破（唐奇安通道、海龟交易法）
  - 动量排序（买过去N月涨最多的）
```

**优点**：简单、不容易过拟合、牛市大赚
**缺点**：震荡市反复止损、滞后于价格、熊市亏钱

---

### 2.2 均值回归

```
逻辑：涨多了会跌，跌多了会涨（万有引力）
持有期：天到周
适合市场：🟡 震荡市
A股有效性：★★★★ 短期非常有效

代表性策略：
  - 布林带回归（触及下轨买，上轨卖）    ← 我们已有
  - RSI 超买超卖（RSI<30买，>70卖）
  - 5日反转（昨天跌最多的今天买）
```

**A股特殊优势**：散户占比高，情绪化交易严重 → 过度反应 → 回归机会多。研究显示 A 股短期反转效应显著强于美股。

**关键发现**（海通证券 2024 研报）：
- A 股「短期反转、长期动量」
- 日/周频：反转占优
- 月频：动量占优
- 反转因子月均 IC 达 2.6%

---

### 2.3 因子选股（多因子模型）

```
逻辑：好公司（便宜+赚钱+成长）长期跑赢
持有期：月
适合市场：所有市场（只要因子有效）
A股有效性：★★★★★ 专业机构主流方法

经典因子分类：
  ┌──────────┬──────────────────┬─────────────────────┐
  │ 因子类别   │ 含义              │ A股代表指标           │
  ├──────────┼──────────────────┼─────────────────────┤
  │ 价值       │ 便宜 vs 贵        │ PE、PB、股息率        │
  │ 动量       │ 近期涨 vs 跌      │ 过去12月涨幅(剔除近1月)│
  │ 质量       │ 好公司 vs 差公司   │ ROE、毛利率、负债率    │
  │ 成长       │ 增长快 vs 慢      │ 营收/利润增速          │
  │ 低波动     │ 稳 vs 飘          │ 历史波动率、Beta       │
  │ 市值       │ 大 vs 小          │ 总市值、流通市值        │
  │ 流动性     │ 换手高 vs 低      │ 换手率、成交量          │
  └──────────┴──────────────────┴─────────────────────┘
```

**怎么用**：
```
1. 每月末计算所有股票的因子值
2. 标准化排名（同行业比较）
3. 多因子加权合成综合得分
4. 买入得分最高的 30-50 只
5. 每月调仓一次
```

**A股有效因子**（2024年券商验证）：
- 市值因子：大市值全年占优（但 9-11 月剧烈反转）
- 估值因子：低估值持续有效
- 动量因子：t-11到t-5月累计涨幅最优
- 波动率因子：低波动持续正超额（ICIR 1.35）

**代表项目**：[GitHub - 1984tkr/multi-factor-stock-selection](https://github.com/1984tkr/multi-factor-stock-selection)（★ 完整多因子选股系统，含IC评估+动态因子剔除）

---

### 2.4 行业轮动

```
逻辑：不同行业在不同经济阶段表现不同，追强势行业
持有期：周到月
A股有效性：★★★★ A股板块轮动特征明显

核心思想：
  1. 先选行业（哪些行业会涨）
  2. 再选个股（行业内最好的股票）
```

**轮动信号**：
| 信号 | 含义 | 数据来源 |
|------|------|------|
| 行业动量 | 过去N天涨最多的行业 | 我们已有（sector_history） |
| 资金流向 | 主力资金净流入 | 同花顺 API |
| 拥挤度 | 过度拥挤 → 反转 | 行业换手率/成交占比 |
| 景气度 | 盈利预期变化 | 分析师一致预期 |

**代表策略**：
- 聚宽「全天候轮动」：大市值好→买大，小市值好→买小，都不好→空仓。2010-2024 回测表现优异
- 国联证券行业轮动 2.0：一致预期景气+龙头效应+动量+反转+拥挤度，年化超额 10.6%

---

### 2.5 动量策略（深度）

```
逻辑：过去涨的继续涨（但要去掉最近涨的——短期反转）

A股最优动量公式（海通证券验证）：
  动量 = 过去12个月涨幅 - 最近1个月涨幅

为什么去掉近1个月？
  → A股短期反转效应很强
  → 近1个月涨的容易回调
  → 去掉后动量因子更纯净
```

**动量失效的三种信号**（会反转）：
1. 市场动能减弱
2. 长短期均收益为负
3. 因子拥挤度高位

三种信号触发后，动量有效月占比仅 28.6%，年化多空 -22.6%。

---

### 2.6 配对交易 & 统计套利

```
逻辑：找两只高度关联的股票，价差偏离均值时套利
持有期：天到周
门槛：需要融券做空（A股困难）

经典配对：
  招商银行 vs 兴业银行  → 银行股联动
  万科 vs 保利          → 地产股联动
  茅台 vs 五粮液        → 白酒股联动
```

**A股障碍**：融券券源少，做空困难。但可以做「只做多价差回归」的简化版。

**代表项目**：[GitHub - tmacsyf/syf_quant-trading](https://github.com/tmacsyf/syf_quant-trading)

---

### 2.7 小市值策略

```
逻辑：小市值股票长期跑赢大市值（规模溢价）
A股有效性：★★★★★ 历史年化可达 50%+

为什么有效：
  - 散户偏好小票炒作
  - 机构覆盖不足 → 定价效率低
  - 壳价值（注册制前）
```

**注意事项**：
- 2024年2月小微盘流动性危机 → 大幅回撤
- 需要严格止损
- 容量有限（资金大就跑不动）
- 聚宽「国九小市值」策略年化 ~100%（回测 2022-2025），但需警惕过拟合

---

### 2.8 事件驱动

```
逻辑：特定事件发生后股价有规律性反应
持有期：天

典型事件：
  - 业绩预告超预期 → 买入
  - 指数调仓（纳入/剔除）→ 纳入前涨、剔除前跌
  - 高分红公告 → 填权行情
  - 龙虎榜游资买入 → 跟庄
```

## 三、A股因子有效性总结

基于 2024 年海通证券、中金公司研报：

| 因子 | A股长期表现 | 2024表现 | 适合策略 |
|------|:--:|:--:|------|
| 市值（小市值效应） | ★★★★★ | 大市值占优（反转） | 小市值策略 |
| 估值（低PE/PB） | ★★★★ | 低估值占优 | 价值选股 |
| 动量（12月-1月） | ★★★★ | 高动量占优 | 趋势跟踪 |
| 反转（短期） | ★★★★ | 短周期有效 | 均值回归 |
| 低波动 | ★★★★ | 持续正超额 | 防御型 |
| 成长 | ★★★ | 唯一未反转因子 | 成长选股 |
| 换手率 | ★★★ | 低换手占优 | 流动性过滤 |

> **核心结论**：A股最稳健的因子是**小市值 + 低估值 + 低波动**三件套。动量需要剔除短期反转。

## 六、对我们系统的启示

### 6.1 哪些方向适合我们

| 方向 | 可行性 | 理由 |
|------|:--:|------|
| **因子选股** | ✅ 最适合 | 月频调仓，计算量可控，有大量研究支撑 |
| **行业轮动** | ✅ 可行 | sector_history 已在积累数据 |
| **波动率策略** | ✅ 可行 | 日频数据够用 |
| **配对交易** | ⚠️ 受限 | 需要做空，¥10,000 资金不够 |
| **小市值** | ⚠️ 受限 | ¥10,000 只能买1-2只，分散不足 |
| **高频/打板** | ❌ 不适合 | 需要实时数据+低延迟，超出框架 |

### 6.2 推荐的下一步：多因子选股

这是最成熟的量化方向，和我们的系统架构天然匹配：

```
现有框架                  只需改造
─────────────────────    ─────────────────
✅ SQLite 日线数据       → 加财务数据表
✅ 300 只成分股          → 因子计算 pipeline
✅ 钉钉推送              → 推送月度调仓清单
✅ 回测引擎              → 因子 IC 分析 + 分层回测
✅ 数据质量保障          → 财务数据校验
```

**最小可行多因子系统**（约 2-3 天开发）：

```
1. 数据：AKShare 下载 PE、PB、ROE 数据入 SQLite
2. 因子：价值(PE)+动量(12月-1月)+质量(ROE) 三个因子
3. 评分：标准化排名 → 等权合成
4. 调仓：月初买 Top 30，等权配置
5. 回测：2019-2026，对比沪深300
```

### 6.3 其他短期可做的

```
🟡 行业动量轮动（利用已有 sector_history）
   → 每周算 5日动量 → 买最强 3 个行业 → 行业内选 Top3 个股

🟡 波动率过滤（改造现有策略）
   → 在现有买入信号上叠加"波动率不过高"条件

🟢 RSI 均值回归策略（新增）
   → RSI(14)<30 买入 → RSI(14)>70 卖出，简单有效
```

## 七、关键教训（来自研报+社区）

1. **没有永远有效的因子**——2019年价值因子惨败，2024年9-11月几乎所有因子反转
2. **回测收益率 > 50% 要警惕**——大概率过拟合或幸存者偏差
3. **简单策略 > 复杂策略**——华尔街验证：3个因子的模型打败了30个因子的
4. **A股择时比选股重要**——熊市里再好的股票也跌
5. **因子拥挤是最大的风险**——大家都在用的因子很快失效
6. **4年以上的回测才有参考价值**——覆盖完整牛熊周期
