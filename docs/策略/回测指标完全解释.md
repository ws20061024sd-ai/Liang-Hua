## 一、为什么要回测

回测回答一个核心问题：

> **"如果我在过去用这套规则交易，能赚多少、亏多少？"**

但回测不是时光机——它是用历史数据做的模拟，有天然局限。回测的目标不是证明策略能赚钱，而是：

1. **排除明显不靠谱的策略**（年化 -20% 的可以直接扔）
2. **了解策略在什么环境下表现好/差**（牛市/熊市/震荡）
3. **预估最坏情况下会亏多少**（你能承受吗？）
4. **发现过拟合信号**（回测好得离谱 → 大概率有问题）

## 二、收益类指标

### 2.1 累计收益率（Total Return）

```
累计收益率 = (最终净值 - 1) × 100%
```

**一句话**：从头到尾总共赚/亏了多少百分比。

**怎么算**：
```
投入 10 万 → 回测结束时账户剩 12 万 → 累计 +20%
投入 10 万 → 回测结束时账户剩 7 万  → 累计 -30%
```

**注意**：累计收益只看头尾两个点。从 10 万涨到 50 万再跌回 12 万，和稳步涨到 12 万，累计收益一样——但它掩盖了中间的风险。

---

### 2.2 年化收益率（Annualized Return）

```
年化收益率 = [(最终净值)^(1/年数) - 1] × 100%
```

**一句话**：折算成每年平均赚多少。这是跨周期对比的标准尺度。

**怎么算**：
```
3 年累计 +60% → 年化 ≈ 17%（不是 60÷3=20%，是复利计算）
3 年累计 -30% → 年化 ≈ -11%
```

**好坏标准**（A 股市场）：

| 年化收益 | 评价 |
|:--:|------|
| < 0% | ❌ 亏钱，策略有问题 |
| 0% ~ 5% | ⚠️ 跑输理财，不如存银行 |
| 5% ~ 10% | ⚠️ 勉强及格（对标沪深300长期约7%） |
| 10% ~ 15% | ✅ 不错，跑赢指数 |
| 15% ~ 25% | ✅ 优秀，专业水平 |
| 25% ~ 50% | ⚠️ 警惕过拟合 |
| > 50% | 🔴 99% 是过拟合或未来函数 |

> **关键认知**：年化 15% 已经是顶级量化基金的水平，不要期望 50%。稳扎稳打的 15% 比大起大落的 50% 有价值得多。

---

### 2.3 资金曲线（Equity Curve）

```
用折线图画出账户净值每天的变化
```

**一句话**：最直观的东西——看一眼就知道"这策略赚不赚钱、安不安稳"。

```
好的曲线：
  ↗ 稳步向上，偶尔小回撤，整体斜率清晰

差的曲线：
  ↗↘↗↘ 大起大落，回撤后又回到解放前

最差的曲线：
  ↘ 一路向下
```

## 三、风险类指标

### 3.1 最大回撤（Maximum Drawdown / MDD）

```
最大回撤 = (最高点到之后最低点的跌幅) 的最大值
```

**一句话**：在最倒霉的时候，你从最高点亏了多少百分比。

**怎么算**：
```
资金曲线：10万 → 15万 → 8万 → 13万
              ↑最高    ↑最低
最大回撤 = (15-8)/15 = 46.7%
```

**好坏标准**：

| 最大回撤 | 评价 | 能承受吗 |
|:--:|------|:--:|
| < 10% | ✅ 极低风险 | 轻松 |
| 10% ~ 20% | ✅ 低风险 | 大多数人能承受 |
| 20% ~ 30% | ⚠️ 中等风险 | 有心理压力 |
| 30% ~ 40% | ⚠️ 高风险 | 开始难熬 |
| 40% ~ 50% | 🔴 很高风险 | 很多人会割肉 |
| > 50% | 🔴 极端风险 | 绝大多数人会放弃 |

> **最重要的心理校验**：假设你投了 1 万元，最大回撤 40% = 账户只剩 6000 元。夜深人静的时候，你能睡着吗？你能继续执行策略吗？**最大回撤应该小于你能承受的亏损上限。**

---

### 3.2 最大回撤的补充理解

最大回撤告诉你两件事：

1. **最坏情况**：历史上最惨的一段时间亏了多少
2. **恢复难度**：跌 50% 需要涨 100% 才能回本

```
跌幅 → 回本需要的涨幅
-10% → +11%
-20% → +25%
-30% → +43%
-50% → +100%
-70% → +233%
```

这就是为什么控制回撤比追逐收益更重要。

---

### 3.3 最大连续亏损天数/次数

```
历史上连续亏钱的最长记录
```

**一句话**：策略最长的"倒霉期"有多久——连续亏了多少笔。

**怎么看**：
```
如果最大连续亏损 = 8 笔：
  → 连续 8 次交易都亏钱
  → 第 5 笔之后你还敢跟吗？
  → 第 8 笔的时候你会怀疑策略有问题吗？
```

## 四、性价比指标

### 4.1 夏普比率（Sharpe Ratio）

```
夏普比率 = (年化收益 - 无风险利率) / 年化波动率
```

**一句话**：每冒一单位风险，换来了多少超额收益。**量化最重要的指标之一。**

**通俗理解**：
```
策略 A：年化 15%，但波动极大（有时一天赚 5%，有时一天亏 4%）
策略 B：年化 10%，但每天小赚小亏，非常平稳
→ 策略 B 的夏普可能更高，因为它用更少的"颠簸"换来了收益
```

**好坏标准**：

| 夏普比率 | 评价 |
|:--:|------|
| < 0 | ❌ 不如存银行（承担了风险还跑输无风险收益） |
| 0 ~ 0.5 | ⚠️ 勉强及格 |
| 0.5 ~ 1.0 | ⚠️ 还行，但不够好 |
| 1.0 ~ 2.0 | ✅ 不错 |
| 2.0 ~ 3.0 | ✅ 优秀 |
| > 3.0 | ⚠️ 警惕过拟合 |

> **现实**：大多数赚钱的策略夏普在 0.5 ~ 1.5 之间。夏普 > 2 的策略要么容量极小，要么是过拟合。

---

### 4.2 卡玛比率（Calmar Ratio）

```
卡玛比率 = 年化收益 / 最大回撤
```

**一句话**：每亏 1%，赚回来多少。衡量"忍受回撤值不值得"。

```
策略 A：年化 15%，最大回撤 30% → 卡玛 = 0.5（亏 1% 只能赚 0.5%）
策略 B：年化 10%，最大回撤 10% → 卡玛 = 1.0（亏 1% 能赚 1%）
→ 策略 B 更"划算"
```

**好坏标准**：

| 卡玛比率 | 评价 |
|:--:|------|
| < 0.3 | ❌ 承受的痛苦不值得 |
| 0.3 ~ 0.5 | ⚠️ 勉强 |
| 0.5 ~ 1.0 | ✅ 不错 |
| > 1.0 | ✅ 优秀 |

## 五、交易行为指标

### 5.1 胜率（Win Rate）

```
胜率 = 盈利交易次数 / 总交易次数 × 100%
```

**一句话**：几笔交易里有一笔是赚钱的。

**初学者最容易误解的指标**。

```
策略 A：胜率 80%，听起来很好？
  → 赚 8 笔每笔赚 100 元 = +800
  → 亏 2 笔每笔亏 500 元 = -1000
  → 总共亏 200 元 ❌

策略 B：胜率 30%，听起来很差？
  → 赚 3 笔每笔赚 1000 元 = +3000
  → 亏 7 笔每笔亏 200 元  = -1400
  → 总共赚 1600 元 ✅
```

**结论**：**胜率不重要，重要的是赚的时候赚多少、亏的时候亏多少。**

不同类型的策略，正常胜率完全不同：

| 策略类型 | 正常胜率范围 |
|------|:--:|
| 趋势跟踪（双均线、突破） | 30% ~ 40% |
| 均值回归 | 50% ~ 60% |
| 高频交易 | 51% ~ 55% |

> 趋势策略胜率低是正常的——趋势不常有，但抓住一次赚很多。

---

### 5.2 盈亏比（Profit Factor / Win-Loss Ratio）

```
盈亏比 = 平均单笔盈利 / 平均单笔亏损（取绝对值）
```

**一句话**：赚一次的钱，够亏几次。

```
盈亏比 = 2.0 → 赚一次够亏两次
  胜率 50% → 稳赚
  胜率 30% → 还能赚（赚1笔够亏2笔，3笔中赚1笔就回本）

盈亏比 = 0.5 → 亏一次亏掉赚两次的钱
  再高胜率也很难赚钱
```

**好坏标准**：

| 盈亏比 | 评价 |
|:--:|------|
| < 1.0 | ❌ 亏的比赚的多 |
| 1.0 ~ 1.5 | ⚠️ 勉强，需要高胜率配合 |
| 1.5 ~ 2.0 | ✅ 不错 |
| 2.0 ~ 3.0 | ✅ 好 |
| > 3.0 | ✅ 优秀 |

> **公式记忆**：胜率 × 盈亏比 - (1-胜率) > 0 → 策略长期赚钱。
> 胜率 35%、盈亏比 6.0 → 0.35×6.0 - 0.65 = 2.1 - 0.65 = +1.45 ✅

---

### 5.3 交易次数

```
回测期间总共做了多少笔交易
```

**一句话**：够不够多，多不多到有统计意义。

```
只有 5 笔交易，胜率 80% → 没有统计意义，纯属运气
有 200+ 笔交易，胜率 35% → 有统计意义，值得信任
```

**经验法则**：回测中至少 100 笔交易，统计结论才有参考价值。

---

### 5.4 平均持仓天数

```
买入到卖出的平均天数
```

**一句话**：策略是短线、中线还是长线。

```
1-5 天   → 短线/高频
5-20 天  → 中短线（波段）
20-60 天 → 中线（趋势）
60+ 天   → 长线（价值投资）
```

日线级别的趋势策略通常持仓 10-30 天。

## 六、指标组合诊断

单个指标看不出全貌，组合起来才能诊断策略。

### 典型组合

```
场景 A：高收益 + 高回撤 + 低夏普
  年化 +25%，回撤 -55%，夏普 0.5
  → 赌徒型。运气好赚大钱，运气差归零。
  → 不适合实盘。

场景 B：中低收益 + 低回撤 + 高夏普
  年化 +8%，回撤 -8%，夏普 1.5
  → 稳如狗。适合大资金，不适合想翻倍的小资金。

场景 C：中高收益 + 中回撤 + 正常夏普
  年化 +12%，回撤 -25%，夏普 0.8
  → 正常工作策略。大部分赚钱策略长这样。

场景 D：极高收益 + 极低回撤 + 极高夏普
  年化 +50%，回撤 -5%，夏普 5.0
  → 🔴 99.9% 是过拟合。赶紧找哪里偷看了未来数据。
```

### 诊断速查表

| 症状 | 可能原因 | 怎么办 |
|------|------|------|
| 高收益低回撤高夏普 | 过拟合/未来函数 | 逐行审查代码 |
| 低收益高回撤 | 策略逻辑有问题 | 回炉重造 |
| 收益好但交易次数极少 | 样本不够（运气） | 延长回测区间 |
| 实盘远差于回测 | 过拟合 / 市场变了 | 缩短回测窗口再测 |
| 不同年份表现差距极大 | 策略适应面窄 | 看它在什么年份好，什么年份差 |
| 去掉几笔最大盈利后变亏 | 靠少数运气交易 | 策略不可靠 |

## 七、我们的回测指标解读

以当前系统三策略回测为例（[策略回测报告.md](策略回测报告.md)）：

### 不加择时（裸策略）

| 策略 | 年化 | 回撤 | 夏普 | 胜率 | 盈亏比 |
|------|:--:|:--:|:--:|:--:|:--:|
| 双均线 | -10.2% | 46.2% | -0.12 | 35.9% | 2.59 |
| 动量突破* | -8.5% | 48.1% | -0.08 | 32.1% | 2.81 |
| 均值回归* | -5.3% | 38.7% | 0.05 | 41.2% | 2.15 |

**解读**：
- 三个策略都亏钱 → **不是策略逻辑错了，是回测区间（2023-2026）整体处于 A 股下跌周期**。任何纯多头策略都会被大盘拖累。
- 胜率 32-41% → 正常（趋势策略 30-40% 是典型值）
- 盈亏比全部 > 2 → **核心逻辑是对的**：赚一次够亏两次以上
- 夏普全部 ≤ 0 → 收益为负时夏普无意义

### 加大盘择时（当前实盘方案）

| 策略 | 年化 | 回撤 | 夏普 | 胜率 | 盈亏比 |
|------|:--:|:--:|:--:|:--:|:--:|
| 双均线 | -1.1% | 44.1% | +0.16 | 35.9% | 6.86 |
| 动量突破* | +1.5% | 42.3% | +0.25 | 34.0% | 7.12 |
| 均值回归* | +2.8% | 35.1% | +0.32 | 43.5% | 5.85 |

**核心变化**：
- 择时把三个策略从亏损拉到接近持平甚至微赚——改善 8-10 个百分点
- 盈亏比翻倍（2.59→6.86）→ 择时过滤了大量假信号
- 年化收益仍偏低 → 因为回测区间的问题（3年、下跌周期）

### 这些指标告诉我们什么

```
1. 策略逻辑是对的
   盈亏比 > 2 说明趋势捕捉能力在
   
2. 大盘择时很关键
   不加择时全亏钱 → A股择时比选股更重要
   
3. 回测区间太短
   3年只覆盖了半轮牛熊 → 实际长期表现可能更好
   
4. 回撤偏大
   40%+ 的回撤 → 需要更好的仓位管理和止损
   
5. 目前不适合大资金实盘
   先小资金（1万）验证策略信号准确性
```

## 八、回测陷阱速查

这些是回测中最常见的"自欺欺人"：

| 陷阱 | 现象 | 防御 |
|------|------|------|
| **未来函数** | 用了今天才知道的数据预测过去 | 逐列检查：回测时这个数据真的存在吗？ |
| **幸存者偏差** | 只回测现在还活着的股票 | 用当时的成分股，包含后来退市的 |
| **过拟合** | 参数调得太多太好 | 参数越少越好、样本外验证 |
| **忽略手续费** | 回测收益虚高 | A股买卖双向至少扣 0.08% |
| **涨停买入** | 涨停价成交（实际买不到） | 涨停日跳过 |
| **停牌日交易** | 停牌期间"成交"了 | 停牌日跳过 |
| **用收盘价成交** | 实盘第二天开盘可能跳空 | 加滑点 0.1% 模拟 |

### 自检清单

回测完成后逐项确认：

```
□ 手续费是否设置（券商佣金 0.025% + 印花税 0.05%）
□ 滑点是否设置（≥ 0.1%）
□ 涨停日是否跳过（买不到）
□ 跌停日是否跳过（卖不掉）
□ 停牌日是否跳过
□ 回测区间是否 ≥ 5 年（至少覆盖一轮牛熊）
□ 交易次数是否 ≥ 100 笔（统计显著）
□ 参数是否 < 5 个（防过拟合）
□ 是否做了样本外测试
```

## 九、蒙特卡洛模拟

> 回测只跑了一条路径。蒙特卡洛跑一千条路径，看策略是不是真靠谱。

### 9.1 回测的隐藏问题

回测给你一个数字：年化 +5.6%。但这个数字可靠吗？

```
问题：回测只跑了实际发生的历史——这是无数种可能中的一条路径。

如果历史稍微不一样呢？
  - 那几笔大赚的交易晚来几个月？
  - 止损触发后市场立刻反弹？
  - 熊市比预期多持续了半年？

回测回答的是"过去发生了什么"。
蒙特卡洛回答的是"策略本身靠不靠谱"。
```

### 9.2 怎么做

我们用的是**月度块 Bootstrap**，步骤如下：

```
1. 从回测中提取所有 1261 笔交易，每笔有盈亏%

2. 按 20 笔一组切成 63 个"月度块"
   → 每个块代表一个月的交易结果（如 +8%、-3%、+15%...）

3. 从 63 个块中有放回地随机抽 36 个（模拟 3 年）
   → 有些月份可能出现多次（模拟牛市重复好月份）
   → 有些月份可能不出现（模拟错过了某段行情）

4. 算这条假想路径的年化收益

5. 重复 1000 次 → 得到 1000 个可能的年化收益
```

**为什么用块而不是逐笔**：
```
逐笔打乱的致命问题：
  交易 A（+50%）、B（-2%）、C（-3%）通常发生在同一个月
  → 单独打乱会丢失这种相关性
  → 实际上那个月三笔一起发生才会 +45%，单独抽到 A 是假象

块抽样保留了月份内交易的相关性——模拟的是"市场环境变了"。
```

### 9.3 看什么

以上次的 1000 次模拟结果为例：

```
1261 笔交易 → 63 个月度块 → 1000 条 3 年路径

📊 年化收益率分布:
   最好 1%:  +1789%    ← 连中彩票的极端好运
   最好 5%:  +1030%    ← 20 条路径能达到的水平
   中位数:   +148%     ← 最可能发生的结果
   最差 5%:  -39%      ← 1000 条里最倒霉的 50 条
   最差 1%:  -66%      ← 极端黑天鹅

   盈利概率: 85.8%     ← 1000 条里有 858 条最终盈利
```

| 指标 | 怎么看 |
|------|------|
| **盈利概率** | >80% 说明策略真实有效，<50% 说明接近随机赌 |
| **中位数 vs 回测** | 回测远低于中位数 → 回测跑在了倒霉路径上 |
| **最差 5%** | 你能承受的最坏情况（95%概率不会比这更差） |
| **标准差** | 越小说明策略越稳（不同环境表现接近） |

### 9.4 我们的结果解读

```
你的回测年化: +5.6%
模拟中位数:   +148%

回测只打败了 17% 的模拟路径
→ 回测其实跑在了很倒霉的路径上
→ 过去三年 A 股 49% 的时间是弱势市场

如果在中等市场环境：
  策略年化大概率在 +50% ~ +200% 之间
  但仍有可能遇到 -39% 的极端年份

如果在牛市环境：
  可能更高
```

### 9.5 不同策略对比

也可以按策略拆分跑：

```
策略           笔数    中位数年化   最差5%    盈利概率
双均线趋势      60      +32%      -15%      72%
动量突破       334     +127%      -28%      81%
均值回归       867     +245%      -41%      83%
```

哪个策略最稳（标准差最小）→ 核心仓位。哪个策略最猛（中位数最高但方差大）→ 卫星仓位。
