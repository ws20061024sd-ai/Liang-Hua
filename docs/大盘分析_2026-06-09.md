## 二、中观：板块与风格

### 2.1 四大板块表现（6/5 数据，6/8 大概率更差）

```
┌──────────┬──────┬─────────┬──────────┬──────────────┐
│ 板块      │ 股票数 │ 涨/跌    │ 平均涨跌   │ 解读          │
├──────────┼──────┼─────────┼──────────┼──────────────┤
│ 上海主板  │ 171只 │ 86涨/79跌│ -0.44%   │ 最抗跌 🛡️    │
│ 深圳主板  │ 75只  │ 33涨/40跌│ -0.72%   │              │
│ 创业板    │ 34只  │ 14涨/20跌│ -1.48%   │ 明显跑输      │
│ 科创板    │ 19只  │ 4涨/15跌 │ -3.16%   │ 崩溃 💀       │
└──────────┴──────┴─────────┴──────────┴──────────────┘
```

**揭示的核心逻辑**：

```
上海主板（银行/保险/建筑/传统工业）
  → 涨跌各半，平均仅跌 -0.44%
  → 资金避风港："卖成长，买价值"

科创板（半导体/芯片/创新药）
  → 19只中15只下跌，平均跌 -3.16%
  → 杀估值最凶的地方
  → 这是典型的"避险模式"
```

### 2.2 股价分层：越贵越惨

```
┌──────────┬──────┬──────────┐
│ 股价区间   │ 股票数 │ 平均涨跌   │
├──────────┼──────┼──────────┤
│ <10元     │ 81只  │ +0.02%   │ ← 基本持平
│ 10-30元   │ 91只  │ -0.36%   │
│ 30-50元   │ 52只  │ -0.22%   │
│ 50-100元  │ 37只  │ -1.65%   │
│ >100元    │ 39只  │ -3.51%   │ ← 暴跌！
└──────────┴──────┴──────────┘
```

**惊人的递进关系**：股价越高，跌得越多。这不是偶然——高价股 = 成长股/机构重仓股 = 这波杀跌的核心目标。低价股 = 银行/建筑/传统周期股 = 资金避风港。

### 2.3 流动性分层：大票跑输

```
┌────────────┬──────┬──────────┐
│ 成交额四分位 │ 股票数 │ 平均涨跌   │
├────────────┼──────┼──────────┤
│ Q1（最低）  │ 75只  │ +0.29%   │ ← 低成交反而涨了
│ Q2         │ 75只  │ -0.12%   │
│ Q3         │ 75只  │ -0.95%   │
│ Q4（最高）  │ 75只  │ -2.42%   │ ← 大资金在出逃！
└────────────┴──────┴──────────┘
```

> 成交最活跃的 75 只股票平均跌 -2.42%，而最不活跃的反而涨了 +0.29%。说明**大量资金在从大票/白马中撤离**，而部分中小市值获得了资金流入。

## 四、综合研判与操作建议

### 4.1 宏观→微观 逻辑链

```
宏观（指数）
  │ 6/3 见顶 → 6/5 破 MA20 → 6/8 加速暴跌
  │ 放量下跌，🟠 弱势
  │
  ├──→ 中观（板块/风格）
  │      │ 上海主板抗跌 (-0.44%)  ← 资金避风港
  │      │ 科创板崩溃 (-3.16%)     ← 资金撤离
  │      │ 低价股基本持平，高价股暴跌 (-3.51%)
  │      │ 大票跑输，小票受青睐
  │      │
  │      └──→ 微观（个股）
  │             四川路桥 +0.96% ← 低价+传统+主板 ✅
  │             江苏银行 +1.25% ← 低价+银行+主板 ✅
  │             大全能源 -9.13% ← 高价+光伏+科创板 ❌
  │
  └──→ 逻辑清晰：
        市场在"卖出成长，买入价值"
        策略信号恰好捕捉到了受益方向
```

### 4.2 当前市场状态

```
┌────────────────────────────────────────────┐
│  🟠 弱势                                    │
│                                            │
│  沪深300 < MA20，短期趋势向下                 │
```

### 4.3 操作建议

```
🔴 短期（本周）：
   - 不跟单买入信号（大盘环境太差）
   - 只观察，不交易
   - 观察钉钉推送的信号，看它们是否在市场下跌中自行消失

🟡 中期：
   - 优先上线防线一（大盘择时）
   - 大盘 < MA20 时系统自动不发买入信号
   - 等市场回到 MA20 上方后再开始跟单

🟢 长期：
   - 这次下跌验证了量化框架的合理性
   - 信号股逆势上涨 → 策略有一定选股能力
   - 但选股能力再强，也需要大盘择时保护
```

### 4.4 如果防线一已启用...

```
6/5 的信号会变成：

  "🟠 大盘弱势（沪深300=4816 < MA20=4880）
   仓位系数=0.0，今日不推荐开新仓
   仅处理现有持仓的卖出信号"

  → 你不会在6/5-6/8下跌中买入任何股票
  → 系统自动替你守住了本金
```
