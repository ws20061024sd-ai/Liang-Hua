## 策略现状（诚实评估）

### 旧三策略：训练轮 ✅ → 淘汰

```
7年回测（2019-2026）：
  A: 无择时    → 年化 +11.9%  回撤 25.3%  夏普 0.80
  B: 二元择时  → 年化  +7.3%  回撤 27.3%  夏普 0.51
  C: 权重匹配  → 年化  +6.8%  回撤 23.4%  夏普 0.48

结论：策略本身赚的是市场Beta而非Alpha。
      最大价值是验证了整套系统架构。
      不会继续优化——直接换多因子。
```
